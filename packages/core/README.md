# @softlaunch/core

The feature flag evaluation engine and config compiler that powers the Softlaunch SDKs. Zero dependencies, runs in any JavaScript environment.

Most applications should use a higher-level SDK instead:

- [`@softlaunch/react`](https://github.com/softlaunch-labs/softlaunch-js/tree/main/packages/react) for React
- [`@softlaunch/js`](https://github.com/softlaunch-labs/softlaunch-js/tree/main/packages/js) for any JavaScript runtime

Use `@softlaunch/core` directly only if you're building a custom integration or evaluating a config blob yourself.

## Install

```bash
npm install @softlaunch/core
```
